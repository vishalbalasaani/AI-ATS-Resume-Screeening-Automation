# AI ATS Resume Screening Automation

AI-powered ATS resume screening workflow built using n8n and OpenRouter APIs.

## Features
- Resume analysis
- ATS score generation
- Automated email responses
- AI workflow automation

## Tools Used
- n8n
- OpenRouter
- Gmail
- Google Sheets
